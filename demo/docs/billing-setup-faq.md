# Billing Setup FAQ

## Required billing state

New Growth and Enterprise workspaces must finish billing setup before usage-based features begin processing jobs. Until billing setup is complete, export and import jobs remain blocked in a preflight state.

## Payment retries

If a payment attempt fails, the workspace enters a grace period. Existing jobs continue for 72 hours, but creating new export jobs may be denied until billing is current.

## Support guidance

When a customer reports a blocked export right after onboarding, confirm that billing setup is complete before troubleshooting the export pipeline itself.
